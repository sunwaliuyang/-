# 3. 安装依赖

安装 yarn

[官网](https://yarnpkg.com/lang/en/docs/install/#debian-stable)

```
curl -sShttps://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add -
echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.
```
